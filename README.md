# Fullstack-E-commerce-web-site-for-computer-peripherals
Fullstack website for computer peripherals [SQL/Java/Hibernate/JSP/Spring Boot]

This project has been done by 1 student from University of Novi Sad, Faculty of Mathematics.

Its done as a project for univeristy.

It presents fullstack E-commerce web site used for computer peripherals.

It has Jpa project that deals with base with Java/Hibernate/SQL and Web project that is done in Spring Boot/JSP.

It's done in Spring Boot using Java with Hibernate, database in done in SQL(MySQL).

Project has all validations and Exception handlers needed.

Maven is used and project has controller/service/repository architecture.

Controllers have CRUD and more complicated requests processed.

Web site has Security config. It has Authentication and Authorisation.

There is client side and admin side. Authorisation handles which resources client has access too and which admin.

Admin has plenty CRUD operations and also can generate PDF reports that are done in Jaspersoft.

Frontend is done with JSP.

Features:
- Spring Boot
- Java
- SQL
- Hibernate
- Security config
- Jaspersoft reports
- JSP
